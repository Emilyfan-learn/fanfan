📜 文言文閱讀(一):魏禧《魏叔子日錄》

【題組資訊】第11-12題

一、📖 完整原文

論小人者,必論其心。小人庸多善事,其心未有無所為而為者。若徒論外事人品真偽,學術邪正,幾不可辨矣。論君子者,又不當徒論其心,心雖純正,而行事偶失,亦即是過。故論小人以心者,所以防閑小人之法;論君子以事者,所以造就君子之方。

二、🔴 核心主旨(紅色標註)

「論小人以心者,所以防閑小人之法;論君子以事者,所以造就君子之方。」

主旨解析:

  • 判斷小人要觀其動機(論心) → 目的:防範小人
  • 成就君子要看其行為(論事) → 目的:培養君子

三、🎨 關鍵字句分色標註

🟡 關鍵概念一:論小人之法

  • 「必論其心」→ 判斷小人要看動機
  • 「庸多善事」→ 小人表面常行善
  • 「其心未有無所為而為者」→ 但動機不純,必有所圖
  • 「幾不可辨矣」→ 只看表面無法辨別

🟢 關鍵概念二:論君子之法

  • 「不當徒論其心」→ 不能只看動機
  • 「心雖純正,而行事偶失,亦即是過」→ 行為有失仍是過錯
  • 「論君子以事者」→ 要從具體行事要求

🔵 對比結構

判斷對象 依據 原因 目的
小人 論其「心」 表面行善,難辨真偽 防範之法
君子 論其「事」 動機純正但行為需盡善 造就之方

五、📝 白話語譯

判斷一個人是否為小人,必須觀察他的動機。小人平常也許做很多善事,但他行為背後的動機,從來不是單純無私的。如果只從表面的行為來判斷人品真假、學術正邪,幾乎無法分辨。

但判斷一個人是否為君子,卻不能只看動機。即使動機純正,但行為上若有疏失,仍然是過錯。所以,從動機來論斷小人,是防範小人的方法;從行為來論斷君子,是培養君子的方法。

六、✅ 考題解析

第11題:君子行事的特質

  • 考點: 成為君子的條件
  • 答案線索: 「心雖純正,而行事偶失,亦即是過」
  • 關鍵理解: 君子必須「動機純正」+「行事無失」兩者兼備

第12題:論小人必論其心的用意

  • 考點: 為何要從動機判斷小人
  • 答案線索: 「小人庸多善事」但「其心未有無所為而為者」
  • 關鍵理解: 不可被小人的表面善行所蒙蔽

📜 文言文閱讀(二):李漁《閒情偶寄·廳壁》

【題組資訊】第20-21題

一、📖 完整原文

石灰壁,最宜銷夏。一白如銀,使暑氣消形遁跡,不敢侵人。人當此際,如立雪山之上,坐冰壺之中,涼氣隨處可掬,透入肌膚,沁人心腑,使人不復有人間煙火之見,真避暑第一良方也。

若遇秋冬之月,則患在太寒,宜加畫卷數幅,古書幾套,懸之壁間,以增文采,兼避冷氣。此又與石灰壁相輔而行者也。然必取古名人之蹟,或今書家可稱者,若俗筆之懸壁也,是代牆壁受辱也。

二、🔴 核心主旨(紅色標註)

「石灰壁,最宜銷夏」+「遇秋冬之月,宜加畫卷...以增文采,兼避冷氣」

主旨解析:

  • 說明石灰壁在夏天與秋冬的不同佈置方式
  • 夏天:純白消暑
  • 秋冬:加掛字畫增添文采並避寒

三、🎨 關鍵字句分色標註

🟡 關鍵概念一:夏日佈置(石灰壁之妙)

  • 「一白如銀」→ 純白色澤如銀
  • 「使暑氣消形遁跡」→ 讓暑氣無處藏身
  • 「如立雪山之上,坐冰壺之中」→ 形容涼爽之感
  • 「真避暑第一良方也」→ 最佳消暑方法

🟢 關鍵概念二:秋冬佈置(加掛字畫)

  • 「患在太寒」→ 問題在於太冷
  • 「宜加畫卷數幅,古書幾套」→ 應加掛字畫
  • 「以增文采,兼避冷氣」→ 增添文化氣息並避寒
  • 「必取古名人之蹟」→ 必須是名家作品

🔴 警示重點

  • 「若俗筆之懸壁也,是代牆壁受辱也」→ 掛庸俗字畫是侮辱牆壁

五、📝 白話語譯

石灰壁最適合用來消暑。純白如銀的色澤,能使暑氣無所遁形,不敢侵襲人。人在此時,就像站在雪山之上,坐在冰壺之中,涼氣隨處可以掬取,透入肌膚,沁入心腑,使人不再有人間煙火的感覺,真是避暑的第一良方啊。

如果遇到秋冬時節,問題就在於太寒冷,應該加掛幾幅畫卷、幾套古書,懸掛在牆壁上,以增添文化氣息,同時避免冷氣。這又是與石灰壁相輔相成的做法。但必須選取古代名人的墨寶,或當今書法家值得稱道的作品,如果將庸俗的字畫懸掛於壁,那是替牆壁受辱啊。

六、✅ 考題解析

第20題:石灰壁的功能

  • 考點: 理解夏日石灰壁的消暑效果
  • 答案線索: 「一白如銀,使暑氣消形遁跡」
  • 關鍵理解: 純白色澤能消暑降溫

第21題:秋冬佈置的目的

  • 考點: 理解加掛字畫的雙重功能
  • 答案線索: 「以增文采,兼避冷氣」
  • 關鍵理解: 既能增添文化氣息,又能避寒

📜 文言文閱讀(三):王安石〈材論〉

【題組資訊】第5題(篇章結構題)

一、📖 完整原文

吾聞之,六國合從,而辯說之材出;劉、項並世,而籌畫戰鬥之徒起;唐太宗欲治,而謨謀諫諍之佐來。此數輩者,方此數君未出之時,蓋未嘗有也。_______________。天下之廣,人物之眾,而曰果無材可用者,吾不信也。

二、🔴 核心主旨(紅色標註)

「人君苟欲之,斯至矣」 ← 此為空格應填入之句

主旨解析:

  • 當君主有需求時,相關人才就會應運而生
  • 天下並非沒有人才,關鍵在於君主是否需要

三、🎨 關鍵字句分色標註

🟢 三個歷史例證(綠色)

  1. 「六國合從,而辯說之材出」

    • 戰國合縱 → 辯士出現

  2. 「劉、項並世,而籌畫戰鬥之徒起」

    • 楚漢相爭 → 謀略家崛起

  3. 「唐太宗欲治,而謨謀諫諍之佐來」

    • 唐太宗治國 → 諫臣聚集

🟡 歸納總結(黃色)

  • 「方此數君未出之時,蓋未嘗有也」 → 君主未出現前,這些人才也不存在

🔴 核心論點(紅色)

  • 空格應填: 「人君苟欲之,斯至矣」 → 君主若有需求,人才自然到來

🔵 結論(藍色)

  • 「天下之廣,人物之眾,而曰果無材可用者,吾不信也」 → 說沒有人才可用,我不相信

五、📝 白話語譯

我聽聞,戰國時期六國實行合縱策略,善於辯說的人才就出現了;劉邦、項羽爭奪天下,善於謀略、勇於戰鬥的人才就崛起了;唐太宗想要治理天下,善於謀略且能直言勸諫的人才就聚集而來。這些人才,在上述這幾位君主尚未出現的時候,也並不存在。當君主有需要的時候,他們就出現了。天下如此廣闊,人物如此眾多,如果說真的沒有人才可以任用,我是不相信的。

六、✅ 考題解析

第5題:篇章結構(填空題)

  • 考點: 理解文章論證邏輯,找出承上啟下的關鍵句

  • 文章結構:

    1. 舉例(三個歷史例證)
    2. 歸納(君主未出時,人才不存在)
    3. ★核心論點(空格處) → 君主有需求,人才就來
    4. 結論(天下必有人才可用)

  • 答案: (A) 人君苟欲之,斯至矣

  • 其他選項為何不適合:

    • (B) 強調古人教育人才 → 偏離主題
    • (C) 強調人才易被發現 → 未呼應君主需求
    • (D) 強調後世君主不任用 → 與全文論點矛盾

📜 文言文閱讀(四):袁宏道〈晚遊六橋待月記〉

【題組資訊】第30-31題(結合白話說明文)

一、📖 完整原文

西湖最盛,為春為月。一日之盛,為朝煙,為夕嵐。今歲春雪甚盛,梅花為寒所勒,與杏桃相次開發,尤為奇觀。

石簣數為余言:「傅金吾園中梅,張功甫家故物也,急往觀之。」余時為桃花所戀,竟不忍去湖上。由斷橋至蘇堤一帶,綠煙紅霧,彌漫二十餘里。歌吹為風,粉汗為雨,羅紈之盛,多於堤畔之草,豔冶極矣。

然杭人遊湖,止午、未、申三時;其實湖光染翠之工,山嵐設色之妙,皆在朝日始出,夕舂未下,始極其濃媚。月景尤不可言,花態柳情,山容水意,別是一種趣味。此樂留與山僧遊客受用,安可為俗士道哉!

二、🔴 核心主旨(紅色標註)

「杭人遊湖,止午、未、申三時;其實湖光染翠之工,山嵐設色之妙,皆在朝日始出,夕舂未下...月景尤不可言」

主旨解析:

  • 批評杭州人只在中午遊湖
  • 真正的美景在「清晨」、「傍晚」、「月夜」
  • 這種雅趣只有山僧遊客能體會

三、🎨 關鍵字句分色標註

🟡 西湖之盛(黃色)

  • 「西湖最盛,為春為月」→ 最美在春天和月夜
  • 「一日之盛,為朝煙,為夕嵐」→ 一天中最美在清晨與傍晚
  • 「今歲春雪甚盛...尤為奇觀」→ 今年梅杏桃同時開,特別奇特

🟢 湖上盛況(綠色)

  • 「由斷橋至蘇堤一帶,綠煙紅霧,彌漫二十餘里」→ 煙霧瀰漫
  • 「歌吹為風,粉汗為雨」→ 歌聲如風,汗水如雨
  • 「羅紈之盛,多於堤畔之草」→ 遊客之多
  • 「豔冶極矣」→ 景色豔麗至極

🔴 作者觀點(紅色)

  • 「杭人遊湖,止午、未、申三時」→ 批評只在中午遊湖
  • 「湖光染翠之工,山嵐設色之妙,皆在朝日始出,夕舂未下」→ 真正美景在清晨傍晚
  • 「月景尤不可言」→ 月夜景色更是美妙難言
  • 「此樂留與山僧遊客受用,安可為俗士道哉」→ 這種雅趣豈能與俗人說

五、📝 白話語譯

西湖最盛大的景色,在於春天和月夜。一天當中最盛的景色,在於清晨的煙霧和傍晚的山嵐。今年春雪特別多,梅花被寒冷所阻,與杏花桃花依次開放,尤其是奇特的景觀。

石簣多次對我說:「傅金吾園中的梅花,是張功甫家舊有的,趕快去觀賞吧。」我當時被桃花所吸引,竟然捨不得離開西湖。從斷橋到蘇堤一帶,綠色的煙霧紅色的霧氣,瀰漫二十多里。歌聲笛聲如同風一般,脂粉汗水如同雨一般,身穿綾羅綢緞的人,比堤岸邊的草還多,豔麗至極。

然而杭州人遊湖,只在午、未、申三個時辰(上午11點到下午5點);其實湖光映照綠色的工夫,山嵐調配色彩的妙處,都在清晨太陽剛出來,傍晚夕陽還未下山時,才最濃郁美麗。月夜的景色更是美妙得無法形容,花的姿態柳的情致,山的容顏水的意韻,別是一種趣味。這種樂趣留給山僧和遊客享受,怎麼可以對俗人說呢!

【題組資訊】第6-8題

一、📖 完整原文

深夜,斜臥床榻,隨手抽取一本小几上疊放的書漫讀助眠,已是長年習慣。書宜輕巧易掌握,內容勿過於深奧嚴肅,否則刺激興奮,反失效果。那夜,從書堆抽出體積特小的日本岩波書店口袋型書,曾閱讀過部分而未竟,書頁間夾著一張書籤,姑且從有書籤的那頁讀起來。翻閱兩三頁後,忽有一張小小的短箋滑下,上面印著淺淺好看藍紫色的鉛印字:

山笑—根據《廣辭苑》

俳句的季節語。謂眾樹一齊吐芽的春季華麗景致。相對的,「山眠」指枯槁失卻精采的山,「山粧」則是被紅葉裝扮的山,各為冬、秋季節語。見於北宋畫家、兼山水畫理論家郭熙的「四時山」。

《廣辭苑》雖未採入,但青青的夏季的山是「山滴」。

這幾行文字與我手中捧讀的書全不相關。把紙片反過來看,正中央印著岩波書店新印製的辭典《廣辭苑》書脊樣本,原來正面文字是為第五版《廣辭苑》做的廣告。

廣告何以題為「山笑」,又引用郭熙「四時山」的字句呢?根據《廣辭苑》,「山笑」一詞,是俳句(日本古典短詩)的季節語,典出於北宋郭熙的《山水訓》。按捺不住好奇,索性走到書房去查究。**《山水訓》的原文是這樣的:「真山水之煙嵐,四時不同。春山淡冶而如笑,夏山蒼翠而如滴,秋山明淨而如粧,冬山慘淡而如睡。」**雖然日文廣告詞引述的文字,把原文的「如笑」、「如滴」、「如粧」、「如睡」改為「山笑」、「山滴」、「山粧」、「山眠」,以適合其語言習慣,且文章次序也略有變動,但郭熙的絕妙比喻,卻被生動地化為一則含帶商業性質的文字裡,不得不令人佩服!至於在四季不同山色中擇取「山笑」為題,從我正在閱讀的書印刷發行時間推斷,應是配合此書是春季版的效果,也是神來之筆。

日本的文士不僅寫作漢詩文,即使和歌、俳句也深受中國文化影響,這段廣告詞可以為證。查得這些來龍去脈,我心中釋然,雖則睡意全消,卻經驗了一次愉悅的失眠。

二、🔴 核心主旨(紅色標註)

「郭熙的絕妙比喻,卻被生動地化為一則含帶商業性質的文字裡...查得這些來龍去脈,我心中釋然...經驗了一次愉悅的失眠。」
